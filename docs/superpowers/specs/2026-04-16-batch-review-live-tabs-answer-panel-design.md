# 批量批改结果页即时流出与答案核对设计

## 背景

当前批量批改链路已经支持异步任务、进度轮询与部分结果返回，但结果页仍然偏“列表型汇总”。在真实班级 PDF 页数较多时，老师有两个新的核心诉求：

1. 哪一页先完成，就先看到哪一页的批改结果，而不是等一批再看。
2. 查看点评时，需要把学生答案图片与点评内容放在一起，方便逐页核对。

这次改动只聚焦结果页交互，不改固定版式单张答题卡批改链路。

## 目标

- 批量任务在处理中时，后端每完成一页就立即更新任务快照。
- 前端结果页实时展示最新完成的页，不等批量窗口聚合。
- 结果页新增页码切换能力，页数较多时能快速定位到指定页。
- 选中某一页后，同屏展示学生答案图与老师点评。

## 方案选择

### 方案 A

按 PDF 原页码顺序展示结果，只更新已完成页的状态。

- 优点：顺序稳定，老师理解成本低。
- 缺点：如果后面的页先完成，老师仍然要滚动很久才能看到。

### 方案 B

按“先完成先展示”的思路即时流出结果，同时提供页码 Tab 做快速定位。

- 优点：最符合“有结果就出”的预期，体验接近 promise 谁先 resolve 谁先显示。
- 优点：通过页码 Tab 保留按原页码核对的能力。
- 缺点：需要前端维护“完成顺序”和“页码索引”两套视图状态。

推荐方案 B。

## 数据设计

### 任务快照

继续沿用 `BatchReviewTaskSnapshot`，并保持以下字段：

- `totalPages`
- `processedPages`
- `result`

### 单页结果扩展

`BatchReviewPageResult` 新增与学生答案图片相关的字段：

- `answerImageObjectKey`: 拆页后学生答案图的对象存储 key
- `answerImageUrl`: 前端可直接展示的图片地址

这样前端不需要再次根据对象 key 发起二次签名请求，轮询拿到任务快照即可直接渲染。

## 后端设计

### 1. Provider 即时推送单页结果

在 `batchVisionProvider` 中保留并发评分，维持当前并发数配置。

每完成一页时：

- 立即把该页结果写入当前进度快照
- 立即触发 `onProgress`
- 不再做短窗口合并

### 2. 结果顺序

任务快照里的 `result.pages` 改为“按完成顺序追加”的数组，而不是严格按页码排序。

同时每个结果保留：

- `pageNo`
- `displayName`

前端可以据此同时支持：

- 最新完成优先查看
- 通过页码 Tab 跳转到原页码对应页

### 3. 学生答案图 URL

Provider 在处理每个拆页对象时，除了给 AI 使用外，还为前端生成可访问的图片 URL，并写入单页结果。

本次优先复用现有对象存储公开访问方式；若后续切回私有读，再增加受控签名接口。

## 前端设计

### 1. 结果页信息结构

结果页拆成三层：

1. 顶部任务状态区
2. 页码切换区
3. 当前页详情区

### 2. 页码切换区

新增横向可滚动 Tab：

- 文案为 `1 / 2 / 3 / ...`
- 已完成页高亮为可查看态
- 未完成页显示“处理中”
- 默认选中“最新完成”的一页

当老师主动点击某页后，保持该选择，不要被后续轮询自动打断。

### 3. 当前页详情区

选中页后展示双区块：

- 学生答案图
- 老师点评

移动端优先采用纵向堆叠：

- 上方图片
- 下方点评

桌面端可用双栏：

- 左边图片
- 右边点评

点评区保留现有字段：

- 得分
- 等级
- 一句话总结
- 做得好的地方
- 主要问题
- 改进建议

### 4. 默认聚焦逻辑

当新的完成页出现时：

- 如果老师尚未手动切换过页签，则自动跳到最新完成页
- 如果老师已经手动切换，则只更新 Tab 状态，不强制切换当前查看页

## 错误处理

- 某页未完成时，详情区显示“正在生成该页点评”
- 若任务失败，已完成页仍保留可查看
- 若某页缺少图片 URL，则展示占位错误卡片，不影响其余页查看

## 测试

### 后端

- Provider 每完成一页就上报一次进度
- 单页结果包含 `answerImageObjectKey` 与 `answerImageUrl`
- 路由保存的任务快照保留已完成页结果

### 前端

- 轮询拿到新页结果后，Tab 会更新
- 默认自动选中最新完成页
- 用户手动切换后，后续轮询不抢焦点
- 当前页同时显示图片与点评

## 实施范围

本次只实现：

- 即时单页流出
- 页码 Tab
- 学生答案图与点评同屏

暂不实现：

- 图片缩放批注工具
- 按学生姓名搜索
- 单页独立分享链接
